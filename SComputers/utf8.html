usage: just write utf8."method name" in you code
allows you to work with the Russian language and other special characters
string operations using utf8 are slower than using string, so it is not recommended to use it just like that

methods:

  • utf8.len(string):number - returns the length of the string
  • utf8.sub(string, i, j):string - it works the same way as string.sub does support Russian and special characters
