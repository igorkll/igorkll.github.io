you can get computer data using getCurrentComputer(), getChildComputers() and getParentComputers()
computer data can be used for direct data transfer between computers
it is a very low-level API and is not recommended for use. it is better to use the setComponentApi or network port and provide access only to the API required for operation
getChildComputers() and getParentComputers() CANNOT SEE computers with the invisibility flag!
getChildComputers() and getParentComputers() return a table with tables containing the data of all connected computers

fields:

  • computerdata.env - direct access to the ENV of the connected computer. you can see all the global variables of this computer
  • computerdata.registers - direct access to the registers of this computer
  • computerdata.crashstate - you can read the computer error message from this table. contains the following fields: hasException, exceptionMsg
