allows you to render voxels in three-dimensional space
please note that holographic projectors for survival mode have limitations that are written in its description
voxel_type - this is the voxel type number. it can be a UUID indicating a custom object that should be displayed
component name - holoprojector


supports the following types of voxels:

  • voxel_type 0 - transparent
  • voxel_type 1 - glowing
  • voxel_type 2 - general block

methods:

  • holoprojector.reset() - resets offset, rotation, scale
  • holoprojector.addVoxel(x, y, z, color, voxel_type:number|uuid|string|nil, localScale:vec3|nil, localRotation:vec3|nil):id - creates a new voxel
  • holoprojector.delVoxel(id:number) - deletes a voxel with the specified ID
  • holoprojector.clear() - deletes all voxels
  • holoprojector.flush() - applies changes
  • holoprojector.setOffset(x, y, z) - changes the displacement of the entire hologram relative to the projector
  • holoprojector.getOffset():x, y, z - returns the value set using holoprojector.setOffset
  • holoprojector.setRotation(x, y, z) - changes the rotation of all voxels relative to the projector
  • holoprojector.getRotation():x, y, z - returns the value set using holoprojector.setRotation
  • holoprojector.setScale(x, y, z) - sets the scale of all voxels
  • holoprojector.getScale():x, y, z - returns the value set using holoprojector.setScale
